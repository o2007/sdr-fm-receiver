# SDR FM Receiver

FM broadcast receiver implemented on Raspberry Pi using GNU Radio and 
SoapySDR.

## Features
- Wideband FM demodulation
- Real-time tuning and gain control
- Audio output via ALSA
- Signal power estimation and squelch

## Structure
- grc/  : GNU Radio flowgraphs
- src/  : control and DSP glue code
- ui/   : user interface (encoders, display)
- docs/ : report and figures
